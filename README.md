# 🐝 BeeFi

> Staking-driven DeFi protocol on **BNB Chain**  
> Designed for **sustainable yield** through dynamic APR

---

## 🚀 Overview

BeeFi introduces a **dual-vault staking system** that balances fixed and dynamic rewards while maintaining long-term token stability.

- ⚖️ Balanced reward structure  
- 🔁 Dynamic APR mechanism  
- 🔒 Secure and predictable withdrawal flow

## 💰 Staking Model

### 🟢 Free Vault
- 4% **fixed APR**
- No lock period
- 48h withdrawal delay

### 🔵 Variable Vault
- 7% ~ 19% **dynamic APR**
- 52-day lock period
- 48h withdrawal delay

> 📈 Dynamic APR is determined by utilization  
> (`totalVariableStaked / REWARD_POOL_BASE`)
